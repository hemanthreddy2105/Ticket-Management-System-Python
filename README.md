# Ticket-Management-System-Python
A Python-based Ticket Management System using OOP concepts, CRUD operations, exception handling, inheritance, and file handling.

The Ticket Management System is a Python-based console application designed to manage customer support tickets. The project demonstrates Object-Oriented Programming (OOP) concepts, CRUD operations, exception handling, inheritance, and file handling.

## Features

* Create Tickets
* View All Tickets
* Search Tickets
* Close Tickets
* Reopen Tickets
* Update Ticket Issues
* Delete Tickets
* Count Open Tickets
* Count Closed Tickets
* VIP Ticket Support
* Data Persistence using File Handling

## OOP Concepts Used

* Classes and Objects
* Constructors
* Encapsulation
* Inheritance
* Method Overriding
* Super Keyword

## Technologies Used

* Python
* File Handling
* Exception Handling

## How to Run

1. Open the project in Spyder or any Python IDE.
2. Run the Python file.
3. Use the menu options to manage tickets.

## Learning Outcomes

* Implemented CRUD operations using OOP.
* Practiced inheritance and encapsulation.
* Implemented exception handling.
* Stored and retrieved data using file handling.
